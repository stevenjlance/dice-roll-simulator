DICE ROLL SIMULATOR

Use this simulator to determine how many rolls it would take to get a certain number of rolls for one side of a six sided dice.

  1. Pick the side of the dice you want to run the progrram for.
  2. Type in how many rolls you want to count for. For example if you tpye in 32, the simulator will tell you how many rolls it will take to get 32 fives (or whatever number you picked) when rolling the dice.

The second simulator checks how long it takes to get a certain number of dice rolls in a row. If you type 10, the simulator will tell you how long it took to get 10 rolls of a five in a row (or some other number).
